# Pricing

**RFP #:** MC-2026-0417
**Section:** §4.5

---

We are proposing a **fixed-fee engagement** at a total of **$47,500**. There are no hourly rates, no not-to-exceed clauses, and no ambiguity about what you will pay. If we find more issues than expected in R1, we fix them — at no additional cost to you.

## Fee breakdown

| Item | Scope | Fee |
|---|---|---|
| **R3 — Test framework & browser tests** | Playwright setup; F1–F4 tests at engagement start; F5 tests after R2 delivery; IT sign-off report | $6,500 |
| **R1 — Reports remediation** | Full audit; defect log; all fixes; written sign-off checklist | $9,000 |
| **R2 — Restocking view** | Backend endpoint; frontend view; operator budget input; edge case handling | $16,000 |
| **R4 — Architecture documentation** | Live codebase review; HTML diagram; IT-ready handoff document | $4,500 |
| **D1 — UI modernization** | Design token refresh; component updates; two direction options presented | $4,500 |
| **D2 — i18n extension** | Extend existing translation layer to remaining views | $3,500 |
| **D3 — Dark mode** | CSS custom-property theme; operator preference persistence | $3,500 |
| **Total** | | **$47,500** |

## What is included

- All work described in the Technical Approach (§4.2) for items R1–R4 and D1–D3
- Written defect log (R1) and test run report (R3) as formal sign-off artifacts
- Architecture documentation in portable HTML format (R4)
- Up to two revision rounds on desired items (D1–D3) based on Meridian feedback

## What is not included

- Hosting, infrastructure, or deployment to production environments
- Database migration (current system uses in-memory mock data — any migration to a production database is out of scope)
- Training or documentation beyond what is specified above
- Work arising from requirements changes after kickoff — addenda are scoped and priced separately

---

## Pricing assumptions

- Fixed-fee pricing assumes the codebase provided with the RFP is representative of current state at engagement start.
- Desired items (D1–D3) are included in the fixed fee; if Meridian elects to descope any of them, the fee is not reduced — the time is already reserved.
- Travel is not required; all work is delivered remotely.
